# edusolve

**edusolve** is an AI-powered learning platform designed to help students solve doubts, understand concepts deeply, and improve through practice — all in one place.

---

## Overview

edusolve allows students to input academic questions and receive:
- Step-by-step solutions  
- Clear and simple explanations  
- Automatically generated practice questions  

Built by a group of four students, edusolve aims to make learning smarter, faster, and more accessible.

## Features

### 1. Doubt Solver
- Input: Text-based questions  
- Output:
  - Step-by-step solution  
  - Easy-to-understand explanation  

---

### 2. Practice Generator
- Generates **5 similar questions** after solving  
- Helps reinforce concepts through repetition  

---

### 3. Community (Coming Soon)
- A space for students to:
  - Ask doubts  
  - Share solutions  
  - Learn collaboratively  

---

### 4. Authentication System
- Secure Login / Signup  
- Personalized dashboard experience  

---

## UI/UX

- Modern **glassmorphism design**
- Soft gradient backgrounds (purple/white)
- Clean and minimal layout
- Smooth animations and premium feel
